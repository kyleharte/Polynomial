# Polynomial

Polynomial is a java library that allows you to manipulate polynomials

## Installation

Simply drag and drop the files into your project and import Polynomial

## Usage
```
Polynomial.read() takes a file as an input and returns a new Polynomial object

Polynomial.multiply() multiplies two polynomials together

Polynomial.evaluate() evaluates a polynomial, takes an input term x

Polynomial.toString() turns the polynomial into a string and returns it
```
